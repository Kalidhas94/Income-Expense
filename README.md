README: Income Expense Calculator

Files: Single HTML file (income-expense-calculator.html)

Description:
A responsive Income & Expense Calculator built with HTML, TailwindCSS (CDN) and vanilla JavaScript.
Features:
- Add income or expense with description and amount
- Edit & Delete entries (CRUD)
- Filters (radio buttons): All / Income / Expense
- Totals displayed at top: Total Income, Total Expense, Balance
- Reset button to clear inputs
- Persists data in localStorage (key: 'ieEntries')
- Responsive layout for mobile and desktop

How to use:
1. Open this file in your browser (double click or drag into browser).
2. Add entries using the form. Choose type (Income/Expense), description, and amount.
3. Use filter radios to show All / Income / Expense entries.
4. Click Edit to load an entry into the form (then click Update Entry to save).
5. Delete removes the entry permanently. Data is saved to localStorage.

Tech stack:
- HTML
- TailwindCSS (CDN)
- JavaScript
